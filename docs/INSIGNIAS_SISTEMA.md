# RUPE — Sistema de Insignias
**Versión:** 1.0 | **Fecha:** Marzo 2026

---

## Especificaciones Técnicas de Insignias

### Dimensiones y Formato
```
Formato principal:  SVG (escalable, sin pérdida de calidad)
Fallback:           PNG

Tamaños requeridos:
  - Miniatura en feed:      32x32 px
  - Perfil del usuario:     48x48 px
  - Vitrina de insignias:   64x64 px
  - Vista detalle:         128x128 px
  - Compartir en redes:    512x512 px

Forma del contenedor:  Hexágono (estilo gaming clásico)
Bordes por nivel:
  - Nivel 1 (Semilla):     Sin borde / borde gris
  - Nivel 2 (Brote):       Borde verde #63B528
  - Nivel 3 (Rama):        Borde ámbar #EF9F27
  - Nivel 4 (Árbol):       Borde morado #7F77DD
  - Nivel 5 (Ecosistema):  Borde degradado dorado animado
```

### Nomenclatura de Niveles de Insignia
```
Nivel 1 → Semilla      (color: gris/verde oscuro)
Nivel 2 → Brote        (color: verde #63B528)
Nivel 3 → Rama         (color: ámbar #EF9F27)
Nivel 4 → Árbol        (color: morado #7F77DD)
Nivel 5 → Ecosistema   (color: dorado animado — solo para los más dedicados)
```

### XP que da cada nivel de insignia
```
Nivel 1 Semilla:       +25 XP
Nivel 2 Brote:         +50 XP
Nivel 3 Rama:         +100 XP
Nivel 4 Árbol:        +200 XP
Nivel 5 Ecosistema:   +500 XP
```
